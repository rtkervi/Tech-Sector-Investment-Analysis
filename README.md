# Tech-Sector-Investment-Analysis

This repository provides investment analysis for major tech companies including Apple, Microsoft, and NVIDIA.

## Features
- Detailed financial analysis
- Comparison metrics
- Visual representation of data

## Tech Stack
- Python
- Pandas
- Matplotlib

## Usage
Clone the repository and run the analysis scripts for insights on tech sector investments.
